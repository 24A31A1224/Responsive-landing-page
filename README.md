# StudySync - Responsive Landing Page

StudySync is a responsive landing page created as **Task 2** for the **KodBud Web Development Internship**. The project demonstrates the fundamentals of responsive web design using HTML5 and CSS3.

## Features

- Responsive layout for desktop and mobile devices
- Modern and clean user interface
- Hero section
- Feature highlights
- Testimonials section
- Footer
- Flexbox-based layout

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries


## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.

## Learning Outcomes

- Improved understanding of HTML5 semantic elements
- Practiced CSS Flexbox for layout design
- Implemented responsive design using media queries
- Built a clean and structured landing page
